## Description

A simple paddle and ball game with sound and visual effects triggered by motion:

- sound effects
- colour changes

Debug menu is at the bottom of the screen. Players are able to customize their gameplay:

- ball speed
- platform size
- sound on/off

## Learning Goals

Learning Goals:

- Designing and implementing a debug screen/menu that fits the game's elements and goals.
- Toggling
